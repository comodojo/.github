# Status: Archived & Read-Only

Hey there! Thanks for stopping by.

Comodojo.org and its repositories are now officially archived. The code is in read-only mode, and I won't be releasing any further updates or fixes.

This was a personal project born out of a genuine love for open source and the belief that code belongs to everyone. I’ve always felt that making source code freely available is the way to drive innovation and democratize technology for everyone.

Unfortunately, life has gotten busy and I no longer have the time to give this pet project the attention it deserves. Instead of letting it gather dust quietly, I’m freezing it here for anyone who might still find it useful.

Thank you to everyone who contributed or simply used my code over the years. You're welcome to fork any of the repos if you want to keep the spirit alive or build something new from them.

Keep building.

Marco Giovinazzi.
